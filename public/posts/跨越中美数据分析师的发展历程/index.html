跨越中美的数据分析师成长之旅

 2022.2.23

大家好，我是张卓，我之前在美国 Wayfair 与 Meta 分别担任数据工程师和数据科学家，2022 年回国加入字节担任高级数据分析师。

Kayla | 领英

“可以简单介绍下你的学校背景以及专业吗？”

我本科于中山大学就读生物科学，大三至大四期间参加双学位课程，专业为管理学院下的创业管理。后来我在美国杜克大学商学院攻读商业分析的硕士课程，花费十个月时间获得了学位。

“为什么本科会选择生物科学这个专业呢？后面又为什么转到其他专业？”

高中的时候我对生物比较感兴趣，当时我以为我是可以成为一个科学家的，并没有考虑到职业发展的问题。因为像这种基础科学一般都要走学术的道路，我的本科同学大概 80% 都去读了硕士或者博士，去年他们刚好博士毕业了，有很多人就继续做科研类的工作或者博士后，极少会进入一些药厂，咨询公司或者投资公司，基本上都是在学术道路上继续走下去。我是在大学读了两年之后，我渐探索清楚了这个专业的职业发展道路，让我知道是自己的兴趣不在这里，我是那种比较耐不住寂寞的人，不太适合独自无聊并且需要完成5年学位的PhD学习，这使我开始转向学习商学，特别是商业分析，我发现这是更加合适的选择，因为它不像市场营销和管理那么纯文科，它融合了理工科的概念，同时也具备多种文化科学的内容

“你为什么会选择去美国，你当初是怎么选择学校和专业的？”

当时我和我的家长都明确地认为自己应该去美国，因为它是一个世界强国，可以获得更多的见识，高等院校也是全世界最多的，会有更多的可选择之处，还有后期的发展潜力也比较好。尽管当时欧元强势，香港也没有现在这样没落。从当今的世界格局来看，去美国比欧洲、澳大利亚、香港都更有优势，如果今天想去探索世界的话，肯定是去美国。选择专业方面，我很感谢我的留学中介，因为他看了我的背景，然后我的兴趣之后，他就给我介绍了这个专业，因为我当时去的时候我是想升那种纯商科的就是marketing或者management那些专业，然后他就跟我说，其实你的理工科背景可以利用起来，然后你也对这些数据比较感兴趣的话，你可以看一下这个专业就是数据分析数据科学，然后当时其实开的也并不多，当时，由于对该专业完全不了解，我也没有想到学校会有数据科学这样一个专业，最初在2016年底到2017年初的时候，开设这个专业的学校便非常少，我们戏称 2018 年是数据分析的元年。

“后来你是如何选择 Offer 的呢？”

对，然后最后是怎么选的？当初 UCB, Columbia 这些项目当年是没有的，然后我当年拿到几个offer里面，一个是比较老牌的 Rochester university 数据分析项目，他已经开了两三年了，然后还有杜克的平安的项目，它其实我去的时候是第一年开做小老鼠那种，然后还有一个是哥伦比亚大学的 applied analytics ，安维txt的项目正在纽约，然后我当时就权衡了一下，就觉得说首先渡客的生活成本肯定是比纽约要低很多的，因为我们这些商客专业都贼贵学费，然后我也不是很有家人和我没有特别的那种名校情节，可能大家会觉得哥大会比较厉害，就在中国的brand会更强一点，但是当时他那个项目其实也是第一年开好像，而且它是开在成人继续教育学院底下的，是那种夜校的感觉。然后生活成本又是最低的，就选了这个队。

“听起来的话，其实你是在因为你刚才讲的你是第一年白老鼠对吧？有一个冒险，或者说在里面你当初是怎么去做风险的评估。”

我就是看他的商学院开了其他项目，比如他们的MBA，然后他另外还有两个master program，然后他们那两个program的就是就业数据都是挺不错的，然后它的位置其实是当时是叫 RTP 三角区，反正也是一个类似于长三角的这种概念，它也是在一个经济发展的比较重点的地带了。然后我就觉得感觉还ok，毕竟杜克大学它也是它首先杜克大学本身的排名比较高，然后它的商学院我就是这个项目开的学院的排名也是挺高的，然后就觉得应该还ok，对。

“然后在10个月过程中你有接触一些或者说学习到一些你知道工作上有用到这些技能吗？因为你现在做的其实都是偏向于数据分析之类的，对吧？这个跟你在学校读的有联系吗？”

完全是在那里学的，他这种项目就是为了就业，所以他教的东西是很实用的，基本上就是你毕了业要用的东西，我到毕业之后还会经常去看以前的PPT，遇到不懂的东西就回看一下学校是怎么教的，然后我觉得他整个项目设置它就是说你就业他也非常看重自己的就业数据，对他讲的东西都是很实用的。

“如果对一些现在他准备要去美国读专业的，或者说他以后想要做data science的，你会有什么建议的一些学习的方式或渠道？”

我觉得是我觉得第一点肯定是要明确你之后工作的时候会需要哪一些技能，然后再倒推你要学什么怎么学。比如说如果现在有一些想学数据科学的同学，不要完全去看学校的课程设置，你要先去看你后面职业的job description，比如说你以后想做Meta 数据分析师，你就去 Meta 官网找一下JD，然后它里面会要求他会列 Responsibility或者 QUanlificitaion，你先从开始看，然后你就会看到说其实他需要的 SQL, Python 技能类的，就是一些技能，然后与此同时他可能会需要去做一些 A/B test ，然后他其实很多很看重对业务产品的理解 product sense，还有你与人沟通的能力，然后这些的话就当你明确了所有这些你要知道的东西，然后你再去看你的课程设置里面是不是都有，然后如果没有的话，你再针对性的去补充。像 SQL，统计学，Python，这些应该是每个项目都会交的。如果他不教的话，其实在网上有非常多的网课，都是在讲这些事情，然后包括 Leetcode 你可以去刷。然后我觉得其实技能类的东西是非常好学非常好，容易上手的。然后但是其实我觉得最难的也是最能划分出一个好的数据分析师一般般的，就是在你的表达沟通和你对业务的理解上面，尤其是我们这种non native speaker，就是在这种表达能力上会会差很多，其实我当时一一去了美国的时候，英语是不好的，所以第一件事情肯定是要把英语弄得非常好。然后所以每天都刷美剧，然后看YouTube，然后也尽量的去跟自己的 Peer 说英语，其实这个项目里80%都是中国人，如果你每天都是对没有很强的意识去锻炼自己的英语的话，你完全可以不懂英语，然后就毕业了。但是这是非常糟糕的一件事情，就是包括在你面试的时候，他其实非常需要你的这种表达能力和业务理解的能力，所以我觉得学好英语口语，然后你的表达能力这个是难的是很难，也很很多人没有这个意识，然后再到对产品的理解，我觉得你去了就一定要去用这些所有的产品，因为我们在国内有墙，并不是所有人都知道 Youtube 具体长什么样，怎么用，它有哪些功能，然后所以说这个人他怎么用，然后你要多去用这些作为一个 User 你自然会有一些感觉，然后包括会可以做一些思维的小训练，你可以想打开一些软件，你想一下他为什么要这个页面为什么是这样的，他每个 Tap 为什么是这样的，然后他做了一些改动之后，比如他要上短视频，你就想一下他为什么要做这个动作，然后你想想完之后，你还可以在网上去看，比如说有一个网站叫 medium 上面会有非常多的这种商业评论的文章，然后也是英文的，然后就可以都去看，然后包括国内有很多公众号，可能也会去讲这些食品类的东西，什么虎嗅，然后那些这些公众号其实都可以去看。我觉得其实。 我觉得还是最重要的是要倒推，就是你要先知道你的office格式，然后你现在缺什么，然后怎么去做，然后每一个东西它的渠道可能是不一样的。

“你有考虑过你因为dis它招的人比较少，你有没有考虑过，如果你找不到合适的专业的工作，你怎么办？你当初有没有考虑转方向或者怎么样的一个方法？”

我当时比较幸运，我们找工作还挺顺利的，我大概毕业前三个月就拿到offer，然后后面三个月就在玩，然后所以就没有太考虑这个问题，然后如果是back up的话，我觉得DS他现在其实有用分化出来一些其他的，比如说什么 data engineer，然后machine learning engineer，其实它都是跟数据科学相关，但又不完全是数据科学的，然后包括他做了一些分化书，其实它title是无限多，就跟data相关的就已经很多，然后这两年为什么会卷？是因为开这个项目的学校越来越多了，就像我刚刚跟你说，我116年申请的时候，整个美国大概就不到6个学校，67个学校很多人我就都升了，但你现在就看每个学校基本都有 data science，所以他的毕业生数量暴增，但是他的岗位的增速肯定是赶不上学位的数量的增速，所以就会感觉比以前要卷。 确实是，但它如果跟 SWE 相比，我觉得应该差不多卷，因为开这个项目的学校也很多，然后他的毕业生也很多，然后码农的话这两年也多招，是因为硅谷在裁员。

“然后你刚刚讲的你的第一份求职是很顺利的，对吧？ 是你刚刚讲可能跟大环境有关对吧？可能多个因素你可以讲一下，你当初有怎么去为求职进行准备吗？然后你感觉效果怎么样？”

我也是走了一些弯路的，就是我们 Program是7月份开始，然后从7月到12月这段时间，我其实想想做的方向是那种生物咨询，7月开始，然后5月毕业。你当初来之前你知道吗？就说你知道ok我来了，这两个月就抽到了，我已经要改好简历要投了。

所以有很多人其实他会在项目开始前，就把这些学校讲的东西都自学一遍，然后秋招的时候就去投，但我当时就没有这样，因为我6月才从中大毕业，然后我7月就要去了。你当初已经知道吗？你当初来之前你知道吗？就说你知道ok我来了，这两个月就抽到了，我已经要改好简历要投了。你当初已经知道吗？而且我们有 career service center，就是学校的职业发展中心，然后他每周都会给你上课，就你7月1去，他就该高就改就教你改简历，然后让你准备那些什么company lister什么的，然后到9月份他就会帮你整理一些公司让你去投了，然后他也会拉一些校友的资源。 我们当时那一届没有校友，因为我们是第一年他会帮我们拉一些什么MBA或者其他项目的的学的学长学姐会来给我们做分享什么的。

“你是从几月份开始真正投简历跟开始面试？”

我就是在9月份的时候跟着大军投开始投简历，但是我当时的方向是错的，我当时简历改的方向是想想去咨询，然后是想去一些生物公司他们要去他们的clients是生物公司，然后我觉得说因为这可以结合我的生物的背景，但是其实我对consulting的理解也非常浅。我之前没有在 consulting 公司学习过，而且 consulting 公司的很多公司都不 sponser 国际生，而且美国人很喜欢做 consulting，竞争比较激烈，所以当时我大概投了反正拿到面试的有两三家，然后基本就是一两轮之后就没有下文了。然后我就反正在寒假之前是没有找到工作的，然后回来寒假我就回国了，回国之后就看到很多朋友其实是在做互联网方面的工作，然后就跟他们都聊了一下，他们就觉得说你其实可以直接投互联网公司，就不需要再去顾及你的生物背景了，其实他们是很包容，他们不会觉得说你本科不是学这个的，你就不可以来互联网公司，然后所以我1月放完假回去之后，我就开始投这种互联网公司，然后 Wayfair 就是其中一家，然后当时就会投完之后就一直面试了三四轮，然后到2月底的时候就拿到offer，然后后面那些就没再面了，反正然后我就觉得挺好的，就在波士顿我很喜欢这个城市。然后作为第一份工作来说，它的 title刚好也是我喜欢比较广，就什么都干一点，就让我感受一下，data analyse 和 data engineer ，他们分别是做什么的，他就融合在一个职位里面，然后我就觉得我挺喜欢的，然后整个面试过程也是让我觉得这个公司挺靠谱的，就刚刚你说怎么样判断一个项目靠不靠谱，其实我觉得在面试过程中你跟面试官的交流，你就看他的网站做得好不好，他的资讯公布的是不是很 solid，我觉得这些都是你能够产生对这个公司或者对这个学校一个判断的一个依据，对，然后反正最后薪酬也ok，然后我就接了这个offer，后面我就没有再面了，当时我投 Meta, 投Google那些大的大厂面试都不给，直接简历就挂了

“这里的话你刚刚提到你第一家公司，它相对来说你能接触不同领域的东西对吧？你之前有实习过吗？”

我有四段实习，第一份实习是在招商银行，然后第二份实习是在在创业公司，在互联网创业公司。然后第三份实习是在尼尔森，就是一家做数据分析的公司，然后最后也在bcg做了一两个月的 Part time assistant。 只有尼尔森是完全相关的，比如说招商银行其实就是做大堂经理什么开户，然后宣传一些理财产品，然后其实跟数据完全没有关系，那个是我第一份实习其实就是一个学长，他想招人帮他干活，然后我就去了，我也没有要报酬，我就是去纯学习的，那个时候我才大二什么都不懂，然后大三大二下学期然后接触到了一家创业公司，是做互联网众筹的，然后也当时其实很流行大学生创业，然后其中一个是华工那边的项目，然后就一起做了互联网，然后这个是我第一次接触互联网行业，就相当于金融行业互联网行业。然后外企就第三个事情是尼尔森外企，然后做数据，其实他是否零售商比较多，反正每个行业都去试了一下，然后最后也试了一下咨询，然后总反正大学时期我去多去搞搞，然后看一下自己的兴趣在哪里这样子。

“接下来的问题是你的第一份工作，因为是你自己的一份正式的工作，对吧？然后当初你可能也没有太多朋友在做这方面的工作，你是怎么去快速的去融入，或者说比较好的去熟悉工作需要的一些技能之类的？”
